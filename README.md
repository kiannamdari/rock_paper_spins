# Rock_paper_spins
this is a game. you can play with bot or second player and your options is three ; rock, paper, and scissors. you choose an option and bot or second player choose an option. game score is 3 if your score is 3 you win. ready play to game?

## Features
single player game
two player game

## Installation
### clone the repo 
git clone https://github.com/kiannamdari/rock_paper_spins.git
cd rock_paper_spins

### create virtual environment
python3 -m venv .venv
#### On linux
source .venv/bin/activate
#### On windows
source .venv/Scripts/activate

### dependencies
pip install pyyaml

## Usage
### run scripts
#### for single plyer game
./single_player_game.py
python single_player_game.py
#### for two player game
./two_player_game.py
python two_player_game.py
